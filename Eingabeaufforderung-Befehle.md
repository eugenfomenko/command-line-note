# CMD-Befehle – Übersicht

Diese Liste enthält wichtige CMD-Befehle mit kurzer Beschreibung, logisch gruppiert für IT-Support und Administration.

---

## Schnellübersicht

| Befehl | Bereich | Kurzbeschreibung |
|--------|--------|-----------------|
| hostname | System | Zeigt den Computernamen an. |
| systeminfo | System | Zeigt detaillierte Systeminformationen. |
| ver | System | Zeigt Windows-Version. |

| ipconfig | Netzwerk | Zeigt IP-Konfiguration an. |
| ipconfig /all | Netzwerk | Detaillierte Netzwerkinformationen. |
| ipconfig /flushdns | Netzwerk | Leert DNS-Cache. |
| ping | Netzwerk | Prüft Netzwerkverbindung zu Ziel. |
| tracert | Netzwerk | Zeigt Route zum Zielhost. |
| nslookup | Netzwerk | DNS-Abfrage durchführen. |
| netstat | Netzwerk | Zeigt aktive Verbindungen und Ports. |

| net view | Netzwerk / Freigaben | Zeigt Computer im Netzwerk. |
| net use | Netzwerk / Freigaben | Verbindet Netzlaufwerke. |
| net share | Freigaben | Zeigt oder erstellt Freigaben. |
| net session | Freigaben | Zeigt aktive Sitzungen auf Server. |
| net file | Freigaben | Zeigt geöffnete Dateien im Netzwerk. |

| net user | Benutzer | Benutzer anzeigen / verwalten. |
| net localgroup | Gruppen | Lokale Gruppen verwalten. |
| net accounts | Benutzer | Kontorichtlinien anzeigen. |

| net time | Zeit | Zeigt Serverzeit / synchronisiert Zeit. |

| net statistics | Analyse | Zeigt Netzwerkstatistiken. |

| query user | Remote | Zeigt angemeldete Benutzer (RDP). |
| logoff <ID> | Remote | Meldet Benutzer ab. |
| rwinsta <ID> | Remote | Trennt Sitzung (RDP reset). |

| attrib | Dateien | Ändert Datei-Attribute. |
| dir | Dateien | Listet Dateien und Ordner. |
| cd | Dateien | Wechselt Verzeichnis. |
